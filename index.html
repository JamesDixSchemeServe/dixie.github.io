Firebase Setup Required

This app uses Firebase Realtime Database for live sync across all players' devices. You need a free Firebase project — setup takes about 5 minutes.

Step 1 — Create a Firebase project

Go to console.firebase.google.com, create a project (any name), then add a Realtime Database in test mode.

Step 2 — Get your config

In Project Settings → Your apps → Add web app. Copy the firebaseConfig object values.

Step 3 — Enter your config below

Config is stored in your browser only. Share the URL with players once connected — they enter the same config on first visit, or you can pre-fill it in the HTML.
